# Документация проекта test_stand

## Описание
Скрипт для отображения потока (стрима) с чередованием видео и изображений из указанной папки. Поддерживается динамическая перезагрузка конфигурации без остановки программы.

---

## Установка

### На Raspberry Pi / Linux

1. Откройте терминал и перейдите в нужную директорию (например, `Desktop`):
   ```bash
   cd ~/Desktop
   ```

2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/petayyyy/test_stand.git
   ```

3. Перейдите в папку проекта:
   ```bash
   cd test_stand
   ```

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

---

## Настройка

### 1. Подготовка медиафайлов
- Создайте папку для медиа (имя задаётся в `config.json`, поле `media_folder`), например `media`.
- Поместите в эту папку изображения и видео.
- Поддерживаемые форматы:
  - **Изображения**: `.jpg`, `.png`, `.gif`, `.bmp`, `.webp`
  - **Видео**: `.mp4`, `.avi`, `.mkv`, `.mov`, `.wmv`, `.webm`
- Порядок воспроизведения определяется алфавитным порядком имён файлов.

### 2. Конфигурационный файл `config.json`
Создайте или отредактируйте файл `config.json` в корне проекта. Пример содержимого:

```json
{
    "stream_url": "rtsp://example.com/stream",
    "stream_seconds": 30,
    "video_seconds": 20,
    "image_display_seconds": 5,
    "media_folder": "media",
    "config_check_interval": 2,
    "display_width": 1024,
    "display_height": 600
}
```

#### Параметры:
- `stream_url` — URL потока (или IP и port для RTSP/HTTP).
- `stream_seconds` — длительность показа стрима (сек).
- `video_seconds` — длительность показа каждого видео (сек).
- `image_display_seconds` — длительность показа каждого изображения (сек).
- `media_folder` — имя папки с медиа (относительно корня проекта).
- `config_check_interval` — интервал проверки изменений `config.json` (сек).
- `display_width`, `display_height` — разрешение окна (например, 1024x600 или 1920x1080). Используется для размера окна и чёрного фона при отсутствии кадра.

## Запуск

1. Убедитесь, что вы находитесь в папке проекта и активировано виртуальное окружение (если используется).
2. Выполните команду:
   ```bash
   python stream_and_video.py
   ```
3. Откроется окно на весь экран.
4. Для выхода нажмите **ESC**.

---

## Особенности работы

- Конфигурация автоматически перечитывается при сохранении изменений в `config.json` (без перезапуска программы).
- Для каждого видеофайла запоминается позиция остановки, поэтому при следующем цикле воспроизведение продолжается с того же места.
- При отсутствии кадра (например, недоступен стрим) отображается чёрный экран.

## Интеграция с Raspberry Pi (иконка на рабочем столе)

Для удобного запуска можно создать ярлык на рабочем столе.

1. Скопируйте файл ярлыка из репозитория на рабочий стол:
   ```bash
   cp ~/Desktop/test_stand/my_stream_app.desktop ~/Desktop/
   ```

2. Убедитесь, что файл имеет права на выполнение:
   ```bash
   chmod +x ~/Desktop/my_stream_app.desktop
   ```

3. При необходимости отредактируйте `my_stream_app.desktop`, указав правильные пути к Python и скрипту.
